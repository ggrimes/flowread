# flowread

A speed reader app designed to help people with dyslexia read more easily.

## Features

- **Spritz-style word-by-word display** with Optimal Recognition Point (ORP) highlighting to anchor focus
- **Adjustable reading speed** (WPM) with a speed slider and keyboard shortcuts
- **Smart pacing** — automatic pauses at sentence and clause boundaries for better comprehension
- **Multiple themes** — Dark, Light, Sepia, and High Contrast
- **File support** — paste text, upload `.txt`, `.pdf`, `.docx`, or `.epub` files, or load from a URL
- **Text-to-speech** — hear each word as it displays
- **Bookmarking** — automatically saves your position and resumes where you left off
- **Highlights** — mark important words with color
- **Chapter navigation** — jump between detected sections
- **Presets** — save named combinations of speed, font, and theme
- **Focus mode** — hides all UI to minimise distraction

## Usage

Open `index.html` in any modern browser. No installation or internet connection required (file loading from URL requires network access).

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Space | Play / Pause |
| ← / → | Skip word back / forward |
| ↑ / ↓ | Increase / decrease WPM by 10 |
| `f` | Toggle focus mode |
| `h` | Add highlight |
| `s` | Open settings |
| `t` | Cycle theme |
